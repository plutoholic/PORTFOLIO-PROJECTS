# SQL Data Analysis Portfolio

## Overview

This repository contains SQL scripts used for exploring, analyzing, and cleaning datasets. The goal of this project is to demonstrate practical SQL skills used in real-world data analysis, including querying datasets, aggregating information, and preparing data for analysis.

## Files in This Repository

### 1. SQLQuery1.sql – Data Exploration

This file contains standard SQL queries used to explore and analyze the dataset.
The queries focus on:

* Retrieving relevant data from tables
* Aggregating data using functions such as `SUM`, `COUNT`, and `AVG`
* Calculating metrics such as percentages and totals
* Grouping and sorting data to identify trends and insights

These queries simulate the type of analysis typically performed by data analysts when examining large datasets.

### 2. SQLQuery2.sql – Data Cleaning

This file focuses on preparing the dataset for analysis by improving data quality.
The cleaning process includes:

* Handling NULL values
* Removing inconsistencies in the dataset
* Standardizing data formats
* Creating new columns when necessary
* Ensuring the dataset is structured and reliable for further analysis or visualization

## Purpose

The purpose of this project is to demonstrate:

* SQL querying and aggregation
* Data exploration techniques
* Data cleaning and transformation
* Preparing datasets for analytics and visualization tools

## Tools Used

* Microsoft SQL Server
* SQL (T-SQL)
